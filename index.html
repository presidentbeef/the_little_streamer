Simple music streaming

Want to stream some music, but don't want to set up a database or deal with importing your music library to yet another application?

The Little Streaming is an extremely no-frills music streaming. Point it at a directory of music and it will start up a Sinatra app. Open your browser to localhost:4567 and stream away.

Using The Little Streamer

The Little Streamer is available as a Ruby Gem:

gem install the_little_streamer
the_little_streamer /my/awesome/music/

Options

Absolutely none.
